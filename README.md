# J.A.R.V.I.S.- Голосовой ассистент

Голосовой ассистент в стиле Тони Старка для Windows. Управление компьютером голосом, запуск программ, поиск в интернете, погода, новости, напоминания и многое другое.

## Возможности

**Голосовое управление**
- Два режима активации: простая (только активация ассистента) и полная (активация + запуск программ)
- Быстрое распознавание речи через Google Speech Recognition
- Голосовые ответы через Google Text-to-Speech (gTTS)

**Управление системой**
- Регулировка громкости (увеличение, уменьшение, выключение звука)
- Блокировка компьютера
- Создание скриншотов
- Выключение и перезагрузка компьютера с возможностью отмены
- Спящий режим

**Мультимедиа**
- Управление медиаплеерами (пауза, следующий трек, предыдущий трек)
- Поиск и воспроизведение музыки через Яндекс.Музыку
- Встроенная музыка при активации (файл backinblack.mp3)

**Приложения и сайты**
- Запуск программ: VS Code, Яндекс.Музыка, Telegram, Discord, браузер, блокнот, калькулятор, проводник
- Открытие сайтов: YouTube, ВКонтакте
- Открытие системных папок: Загрузки, Рабочий стол, Документы, Музыка

**Информационные сервисы**
- Погода для любого города (по умолчанию Балашов)
- Последние новости из RSS-лент
- Текущее время и дата
- Системная информация (заряд батареи, загрузка процессора, использование ОЗУ)

**Дополнительные функции**
- Напоминания с сохранением в файл
- Буфер обмена (копирование и вставка текста)
- Генерация безопасных паролей
- Шутки и ответы на общие вопросы

## Требования

- Операционная система Windows 10 или Windows 11
- Python 3.8 или новее
- Микрофон
- Подключение к интернету (для распознавания речи, голосовых ответов, погоды и новостей)

## Установка

### Автоматическая установка

1. Скачайте установщик JARVIS_Setup.exe
2. Запустите файл и следуйте инструкциям
3. При необходимости установщик предложит установить Python и необходимые библиотеки

### Установка из исходного кода

1. Клонируйте репозиторий:
   git clone https://github.com/ВАШ_ЛОГИН/JARVIS.git
   cd JARVIS

2. Установите зависимости:
   pip install -r requirements.txt

3. Запустите ассистента:
   python JARVIS.py

## Зависимости

Все зависимости указаны в файле requirements.txt:

- speechrecognition - распознавание речи
- gtts - синтез речи (Google Text-to-Speech)
- pygame - воспроизведение звука
- pyautogui - управление клавиатурой и мышью
- psutil - системная информация
- requests - работа с API и RSS
- pyperclip - работа с буфером обмена

## Настройка перед использованием

### API ключ для погоды

Для работы погоды требуется бесплатный API ключ OpenWeatherMap:

1. Зарегистрируйтесь на https://openweathermap.org
2. Подтвердите email
3. Получите API ключ в личном кабинете
4. В файле JARVIS.py найдите класс Config и замените значение WEATHER_API_KEY:

   WEATHER_API_KEY = "ваш_ключ"

### Пути к приложениям

В файле JARVIS.py в классе Config укажите пути к вашим программам в словаре APP_PATHS:

   APP_PATHS = {
       "vscode": r"C:\Путь\к\Code.exe",
       "yandex_music": r"C:\Путь\к\YandexMusic.exe",
       ...
   }

### Музыка при активации

Поместите файл backinblack.mp3 в папку с программой или укажите другой путь в переменной MUSIC_FILE.

## Известные ограничения

- Для работы распознавания речи и голосовых ответов требуется интернет
- Функции pyautogui (управление клавиатурой и мышью) работают только на Windows
- При активации через "Просыпайся, папочка вернулся" запускаются программы, указанные в FULL_START_APPS
- Для работы погоды необходим API ключ OpenWeatherMap

## Устранение неполадок

**Микрофон не найден**
- Проверьте подключение микрофона
- Убедитесь, что микрофон выбран как устройство ввода по умолчанию
- Проверьте разрешения на доступ к микрофону в Windows

**Ошибка Tcl/Tk при запуске**
- Добавьте в начало файла строки с установкой переменных окружения TCL_LIBRARY и TK_LIBRARY
- Или переустановите Python с отмеченной опцией "tcl/tk and IDLE"

**Не распознаётся речь**
- Проверьте интернет-соединение
- Убедитесь, что микрофон работает
- Попробуйте говорить чётче и громче

**Не работает погода**
- Получите бесплатный API ключ на openweathermap.org
- Вставьте ключ в переменную WEATHER_API_KEY в классе Config
