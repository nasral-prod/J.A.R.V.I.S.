## 🎯 ОПИСАНИЕ

J.A.R.V.I.S. — это голосовой ассистент для Windows, вдохновлённый искусственным
интеллектом Тони Старка. Управляйте компьютером голосом: запускайте программы,
регулируйте громкость, получайте погоду и новости, создавайте напоминания,
генерируйте пароли, делайте скриншоты и даже общайтесь с нейросетью.
